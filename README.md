# QuoteVerse

QuoteVerse is a **full-stack quotes web app** built with **Next.js, TypeScript, and Node.js**. It provides a **secure API**, **interactive frontend**, and **reusable hooks** for state management.

## Features

- 🎉 Next.js with App Directory
- ⚛️ React & TypeScript
- 💨 Responsive and interactive frontend
- 🔗 REST API for random quotes
- 🧩 Reusable hooks
- 🛡️ API key validation middleware
- ⚡ Modular and maintainable architecture
